# KOLUMN

Skill-based arcade puzzle game. Drop, rotate, match.

**[Play KOLUMN](https://constarik.github.io/kolumn/)**

## Mechanics
- 5×5 grid, 5 symbol types
- Rotate strip, match column tops → clear
- Timer multiplier: ×10.0 → ×1.0 (18 sec)
- Horizontal chains: 3+ same tops → bonus clear
- Bomb: once per game emergency rescue

## Controls
- ← → : Rotate strip
- SPACE : Drop
- P : Pause
